XRPL Payment Flow
Group Creation
Host sends stake XRP → group treasury (XRPL Payment tx). Treasury address stored on Supabase + XRPL ledger.
Members Join
Each participant sends their stake XRP via direct Payment tx. Confirmed in <5s. No intermediary.
Violation Detected
AccessibilityService fires → app triggers penalty Payment from violator's wallet to treasury. Real XRP, real consequence.
Session End
Host distributes treasury to clean participants. Roadmap: Ledger Nano signs automated EscrowFinish.
